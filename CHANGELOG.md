# Changelog

All notable changes to FilmSEC will be documented in this file.

## [1.0.0] - 2024-12-28

### Added
- Initial release of FilmSEC - Mood-based Movie Recommendation Platform
- User authentication system with JWT tokens
- Movie browsing with mood-based filtering (Happy, Sad, Excited, Relaxed, Thoughtful, Romantic)
- Pagination system (50 movies per page)
- AI chat assistant for personalized movie recommendations
- Support for both English and Azerbaijani languages
- AI assistant with transliteration and typo handling for Azerbaijani
- Movie favorites and watch later functionality
- Movie comments and reviews system
- Community discussions and replies
- Advanced filtering (type, language)
- Responsive design for all devices
- Dark mode support

### Technical
- Backend API built with Node.js, Express, TypeScript
- Frontend built with React, TypeScript, Vite
- Database: Supabase (PostgreSQL)
- Authentication: JWT-based
- Deployed on Render.com (backend) and Vercel (frontend)
